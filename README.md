# ⛰️ GRASKAMA · Open Recruitment Pramuka SKAMA

Website resmi untuk penerimaan anggota baru **Pramuka SKAMA (GRASKAMA)** Ambalan Soekarno – Fatmawati, SMK Negeri 1 Maesan. Dirancang sebagai portal pendaftaran digital yang modern, responsif, dan terintegrasi dengan Google Sheets untuk pengelolaan data pendaftar.

---

## 📌 Tentang Proyek

GRASKAMA adalah platform pendaftaran online untuk kegiatan Open Recruitment Pramuka di lingkungan SMK Negeri 1 Maesan. Website ini menggabungkan nuansa petualangan alam dengan tampilan antarmuka yang elegan dan profesional, cocok untuk acara berskala nasional.

Fitur utama:
- **Hero Section** dengan latar gunung, hutan, kabut, dan bendera Indonesia.
- **Formulir pendaftaran** yang terhubung langsung ke Google Sheets.
- **Dashboard admin** dengan perlindungan password untuk melihat data pendaftar.
- **Countdown** batas pendaftaran (1 Agustus 2026).
- **Galeri kegiatan** (dengan placeholder gambar).
- **FAQ**, **timeline**, dan **kontak media sosial**.

---

## 🚀 Fitur Unggulan

| Fitur | Deskripsi |
|-------|-----------|
| 📝 Formulir Pendaftaran | Nama, NISN, Kelas, Jurusan, Gender, Tanggal Lahir, Alamat, Telepon, Instagram, Riwayat Kesehatan, Motivasi, Upload Foto. |
| 📊 Tabel Admin | Lihat semua data pendaftar dalam bentuk tabel (password: `graskama2026`). |
| ⏳ Countdown | Hitung mundur menuju penutupan pendaftaran (1 Agustus 2026). |
| 🖼️ Galeri Kegiatan | Menampilkan foto-foto kegiatan pramuka (placeholder). |
| 📞 Kontak | WhatsApp, Instagram, TikTok, Google Maps (dengan ikon berwarna). |
| 🌓 Dark Mode | Mendukung tema gelap otomatis sesuai preferensi perangkat. |
| 📱 Responsif | Tampilan optimal di desktop, tablet, dan ponsel. |

---

## 🛠️ Teknologi yang Digunakan

- **HTML5** – Struktur halaman.
- **CSS3** – Styling dengan variabel CSS, efek glassmorphism, dan animasi.
- **JavaScript (ES6)** – Interaktivitas, fetch API, countdown, dan kontrol admin.
- **Google Apps Script** – Backend sederhana untuk menyimpan data pendaftar ke Google Sheets.
- **Google Fonts** – Bebas Neue, Cinzel, dan Poppins.
